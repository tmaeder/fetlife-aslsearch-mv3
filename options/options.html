PAT-FetLife dataset

URL of a JSON file containing reported abuse profiles. Format: array of {userId, reason, url}. Leave empty to disable warnings.

Home location (for distance filter)

Sets a reference point. Distance column and "Within km" filter become active when this is set.

Note vault (encrypts private notes at rest)

When enabled, your private notes are encrypted with a passphrase you choose (AES-GCM, PBKDF2-SHA-256, 210k iterations). The unlock key is held only in chrome.storage.session and is wiped when you close Chrome. Forgetting the passphrase makes existing notes unreadable. Other lists (seen / blocked / pinned) remain plaintext.
